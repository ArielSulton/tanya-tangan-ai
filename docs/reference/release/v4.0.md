# PENSyarat AI v4.0 — Visual Kata Keterangan + Quiz Rework

> **Release**: 22 April 2026
> **Codename**: TanyaTangan Visual 2
> **Stack**: Next.js 15 (App Router) + FastAPI + PostgreSQL (Supabase) + Drizzle ORM

---

## Apa yang Berubah dari v3.0

v3.0 memperkenalkan platform kosakata visual dengan gestur SIBI. v4.0 melakukan **rework total pada mode kuis** dan **pengalaman visual kata keterangan**.

### Perubahan Utama

| Area | v3.0 (Sebelumnya) | v4.0 (Sekarang) |
|---|---|---|
| Quiz kata_keterangan | Emoji drag-and-drop ke drop zone | **Visual card display + tombol pilihan kata** |
| Clue di quiz | Emoji + teks label di atas visual | **Visual saja, tanpa teks clue** |
| Kartu kata_keterangan | Mixed styling antar sub-tipe | **Style unified** (3/4 cards → ConcreteWordCard style) |
| Immediate suggestion | Tidak ada | **y→yang (kata_keterangan), k→kelinci (hewan)** |
| Pending gesture UI | Tidak ada | **"Kata Terdeteksi" muncul di area input** |
| Hand pose J | J tersedia sebagai huruf SIBI | **J diganti dengan Y** |

## Fitur Baru v4.0

### 1. Quiz Rework — Visual Card + Word Choice

Mode kuis `/vocab/kata_keterangan` sepenuhnya di-rewrite dari emoji drag-and-drop menjadi **visual card display dengan tombol pilihan kata**.

#### Sebelum (v3.0)
- DndContext dengan DraggableItem (emoji) dan DropZone
- Emoji grid sebagai opsi jawaban
- Drag gesture ke drop zone
- Immediate visual feedback

#### Sesudah (v4.0)
- `DragDropQuiz` menampilkan satu visual card per soal
- 4 tombol pilihan kata (2x2 grid) menggantikan drag-and-drop
- Visual card ditentukan oleh `visualType` pada `KATA_KETERANGAN_QUIZ`:
  - `size-contrast` → `<SizeContrastCard>`
  - `intensity` → `<IntensityCard>`
  - `selection` → `<SelectionCard>`
  - `combination` → `<CombinationCard>`

**Data quiz (`KATA_KETERANGAN_QUIZ`):**

| Word | Visual Type | Choices |
|---|---|---|
| sangat | intensity | sangat, besar, yang, dan |
| besar | size-contrast | sangat, besar, kecil, dan |
| kecil | size-contrast | sangat, besar, kecil, yang |
| yang | selection | yang, dan, kecil, sangat |
| dan | combination | dan, yang, besar, sangat |

### 2. Mode Prop pada Kartu Visual

Setiap kartu visual menerima prop `mode?: 'explore' | 'quiz'` (default: `'explore'`).

**Quiz mode** — semua teks clue disembunyikan:
- Heading kata (contoh: "BESAR", "Biasa+Sangat")
- Footer sign description (contoh: "Isyarat BESAR (Membuka)")

**Explore mode** — label penuh tetap terlihat seperti v3.0.

Komponen yang mendukung `mode`:
- `SizeContrastCard` — word heading + "Isyarat {WORD} ({Membuka|Menguncup})" footer
- `IntensityCard` — "Biasa"/"Sangat" headings + "Isyarat SANGAT" footer
- `SelectionCard` — "Apel {YANG} hijau" heading + "Isyarat YANG" footer
- `CombinationCard` — "Apel {DAN} Anggur" heading + "Isyarat DAN" footer

### 3. Style Unification — Kartu Kata Keterangan

3 dari 4 kartu kata_keterangan diseragamkan ke gaya `ConcreteWordCard`:
- `rounded-2xl`, `border border-slate-100`, `shadow-md`
- Padding konsisten `p-6`
- Font heading dihapus di quiz mode

**IntensityCard tetap dipertahankan** dengan desain split complex (atas-bawah) karena diminta user.

### 4. Immediate Suggestion (EXPERIMENTAL)

Fitur eksperimental di `GestureRecognition.onWordFormed` (`frontend/src/app/vocab/[kategori]/page.tsx`):

```tsx
// EXPERIMENTAL: Jika input startsWith 'y' di kata_keterangan, langsung suggest 'yang'
const immediateSuggestion =
  normalized.startsWith('y') && kategori === 'kata_keterangan'
    ? 'yang'
    : normalized.startsWith('k') && kategori === 'hewan'
      ? 'kelinci'
      : null
```

**k→kelinci untuk hewan** — awal huruf K langsung suggest kucing (kemudian diubah ke kelinci sesuai request).

### 5. Pending Gesture UI — "Kata Terdeteksi"

Ketika gestur terdeteksi, UI "Kata Terdeteksi" muncul **di area input** (bawah kamera), bukan di area hasil. Idle state tetap tampil berdampingan dengan pending state.

Komponen: tetap di `page.tsx` (bukan komponen terpisah).

### 6. J → Y Hand Pose Replacement

Huruf J dihapus dari konfigurasi SIBI dan diganti dengan Y:

**`frontend/src/lib/ai/config/sibi-config.ts`:**
- `J` dihapus dari array `ALPHABET`
- `J` dihapus dari array `AVAILABLE_GESTURES`

**`frontend/src/components/handsigns/index.ts`:**
- `jSign` dihapus dari exports

File `Jsign.ts` di `handsigns/` tetap ada (tidak dihapus).

## Alur Pengguna (User Flow)

### Siswa (Public, Tanpa Login)

```
Landing Page → /vocab (kategori grid)
  → Pilih kategori → /vocab/[kategori]
    → Mode Jelajah:
      → Nyalakan kamera → gestur SIBI → "Kata Terdeteksi" muncul
        → Cari → hasil kata ( ConcreteWordCard / visual card / adverb component )
        → Tidak ditemukan → AI Fallback (3-layer correction)
      → Input manual (keyboard) → hasil kata
    → Mode Kuis (toggle button):
      → kata_keterangan → VisualCard + 4 tombol pilihan kata
        → Klik jawaban → feedback (correct/wrong + confetti)
        → 5 soal → Lihat Hasil
      → lainnya → ImageMatchQuiz (gambar drag & drop)
```

## Konfigurasi Penting

| Variabel | Nilai | Catatan |
|---|---|---|
| `ENVIRONMENT` | `development` / `production` | Production menonaktifkan Swagger dan mengubah CORS |
| `BACKEND_URL` | `http://localhost:8000` (dev) / `http://backend:8000` (docker) | Proxy Next.js → FastAPI |
| `GROQ_API_KEY` | diperlukan | Untuk Layer 3 LLM correction |
| `DATABASE_URL` | Supabase PostgreSQL | Drizzle ORM (frontend) + SQLAlchemy (backend) |
| `PINECONE_API_KEY` | tidak digunakan | v3.0+ tidak menggunakan RAG |

### Perhatian

- Frontend `next.config.ts` memiliki `typescript.ignoreBuildErrors: true` — **selalu jalankan `bun run tsc` manual** sebelum mengklaim frontend selesai
- Backend `config.py` membangun SQLAlchemy URL dari variabel individual (user, password, host, port, dbname), bukan `DATABASE_URL` langsung
- `SECRET_KEY` backend harus ≥32 karakter
- `.gitignore` mengecualikan file model `.h5` dan `.bin` kecuali `model.h5`

## Perintah Penting

```bash
# Start semua service (development)
COMPOSE_BAKE=true docker compose -f compose.dev.yaml up --build

# Frontend only
cd frontend && bun run dev       # Turbopack dev server (port 3000)
cd frontend && bun run tsc        # Type-check WAJIB sebelum klaim selesai
cd frontend && bun run lint        # ESLint

# Backend only
cd backend && source .venv/bin/activate
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

# Database migration
cd frontend && bun run db:generate   # Generate migration
cd frontend && bun run db:migrate    # Apply migration
cd frontend && bun run db:studio     # Drizzle Studio GUI

# Seed vocab data
cd backend && python scripts/seed_vocab.py
```

## Constraint Desain

- **Tanpa audio** — semua feedback untuk siswa SDLB-B harus visual/haptic (confetti, warna, emoji animation)
- **Tangan saja** — tidak ada NMM Face Mesh detection, hanya gestur tangan SIBI
- **Quiz tanpa teks clue** — mode kuis menyembunyikan label kata untuk menjaga kesulitan
- **Indonesian language** — platform khusus peningkatan bahasa Indonesia, tidak per-mapel
